## What's New in v1.3.0

- **Multi-file Support** — Now host CSS, JS, and JSON files alongside your HTML for full web project previews.
- **Expanded File Compatibility** — All file types including `.txt`, `.json`, and `.xml` are now served via localhost.
- **Stability Fix (WakeLock)** — Added WakeLock to prevent Android from killing the background server process.
- **UI Title Bar Fix** — Corrected the title bar positioning to ensure a consistent look across all Android versions.
- **Connection Fix** — Resolved custom port issues so viewer devices now connect seamlessly on the correct port.
- **Support Development** — Integrated Google AdMob with a short ad before hosting to keep development free.
- **Enhanced UI** — Cleaner layout and improved status indicators for a better user experience.
